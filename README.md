# OpenUI
一个关于 Android 应用开发 UI 的个人收集仓库，用于非盈利的教育目的。

声明：本项目仅供学习交流，如作他用所承受的法律责任一概与作者无关。

## 格式 
大小统一为： `72px * 72px`    
格式统一为： `.png`     
资源文件建议存放于 `drawable-hdpi` 下
